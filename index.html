Services

From the back end to the front end, website development is my specialty.

Microsoft ASP.NET

Microsofts server-side web application framework ASP.NET Core is the underlying service that drives a website. I've crafted my career around ASP.NET technology and its become a part of whom I am over several years of development. All in all ASP.NET has been an absolute pleasure to work with.

Docker

Docker is a useful tool that I use to wrap up web applications into containers before deploying them to a cloud host. Docker containers are, in a sense, mini computers that house and serve websites. These websites can be scaled up rapidly to meet demand or scaled down to save resources.

Cloud Services

Cloud computing is not a new technology, but how it works with web applications has significantly improved over the past few years. Using the cloud, you're able to use tools such as scalable container services, giving you greater control over how your website is scaled and distributed.

Google Angular

Developing with Angular is so much fun. I've used Angular since the early days of AngularJS and watched the framework evolve into its own platform. Angular is an MVC framework utilizing TypeScript which gives us developers a powerful toolset to build superior, interactive applications.

Facebook React

React is another great library that I like to incorporate into my projects when Angular is too much power for the needs of the application. React lacks the MC in MVC (without plugins) but it's a great go-to for quick, reusable UI components. I prefer Angular over a plugged React for SPA's.

HTML5 / CSS3

Ah, good old HTML and CSS. We all know or have at least heard of this technology (unless you live under a rock) but I feel it's still worth mentioning as it's the basic skill of any web developer. I always design my landing pages with a mobile first approach and with pixel perfect precision!

My story as a developer

Since beginning my journey in software development over 15 years ago, I've had the pleasure of working with a vast amount of technologies from game and mobile development to full stack web design. During this time I've become accustom to a wide variety of development concepts and strategies that have allowed me to become an effective developer across many technologies.

From the back-end to the front-end, I'm very passionate about web design in all aspects and I continue to push new boundaries with it every day. Keeping up with latest and greatest in web innovation and technology, I'm always eager to apply new concepts and functionality that is both fun for me to develop and great for the consumer.
